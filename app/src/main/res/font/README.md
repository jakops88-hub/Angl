# Font Files Directory

This directory contains the local TTF font files for the Angl app's luxury design system.

## Required Font Files

Place the following font files in this directory:

1. **cinzel_bold.ttf** - Elegant serif font for headings
   - Download from: https://fonts.google.com/specimen/Cinzel
   - Weight: Bold (700)

2. **montserrat_medium.ttf** - Modern sans-serif for body text
   - Download from: https://fonts.google.com/specimen/Montserrat
   - Weight: Medium (500)

3. **montserrat_bold.ttf** - Bold weight for emphasis
   - Download from: https://fonts.google.com/specimen/Montserrat
   - Weight: Bold (700)

## How to Add Fonts

1. Download the font files from Google Fonts
2. Place the TTF files directly in this `res/font/` directory
3. Ensure filenames match exactly:
   - `cinzel_bold.ttf`
   - `montserrat_medium.ttf`
   - `montserrat_bold.ttf`
4. Rebuild the project

The app will automatically use these fonts for the Dark Luxury design system.
